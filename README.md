HR Attrition SQL Analysis
📌 Project Overview

This project analyzes employee attrition patterns using SQL. The goal is to identify high-risk segments, departments, and roles contributing to employee turnover.

🛠 Tools Used

MySQL

MySQL Workbench

SQL (Aggregation, CASE, Window Functions)

📊 Key Insights

Highest attrition observed in Under 30 age group

Low income employees show highest turnover rate

Sales Representative role ranked #1 in attrition

Significant variation across departments

🧠 Analysis Performed

Overall attrition rate calculation

Department-wise attrition comparison

Age segmentation analysis

Income band analysis

Job role ranking using window functions
Business Value

This analysis helps HR teams:

Identify high-risk employee segments

Improve retention strategies

Target early-career employees

Optimize workforce planning
